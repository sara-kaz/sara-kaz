## Featured Project

### 🩺 M.S. Thesis — Hardware-Accelerated Deep Learning for Multimodal Biomedical Monitoring
*Patent Pending · C++ / Python · ESP32-S3*
- Multi-task CNN-SE-Transformer running **entirely on-device** for real-time ECG arrhythmia, stress, and activity classification — no cloud dependency.
- On-device adaptive learning via head-only backpropagation (4,984 trainable vs. 107,568 frozen params) → **715× reduction** in activation storage over full backprop.
- Persists personalized weights to NVS flash across reboots within 300 KB heap / 8 MB PSRAM.
- **297 ms inference latency** on 10-second multi-sensor windows (ECG, PPG, IMU, GSR).
